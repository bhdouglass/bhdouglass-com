Remindor Apps

Indicator Remindor

Remindor Qt

Built for people that want more than an alarm clock, the Remindor Apps are designed to give the user fully customizable and repeatable reminders. The Remindor apps are more than alarm clocks because they allow you to express dates and times in plain English (and other languages to a lesser extent) and they allow reminders to repeat in many different ways. Reminders can pop up a notification, play a sound, run a command, or even send reminders to your phone (via Boxcar or Pushbullet). All this wrapped in an easy-to-use interface.
There are many situations in which you'll find Indicator Remindor useful: you can use it to set reminders about important events, to get a notification when your favourite TV show is about to start or to wake you up in the morning, etc. Alin Andrei on WebUpd8

Install Indicator Remindor

Ubuntu 12.04, 14.04, & 14.10

sudo apt-add-repository ppa:bhdouglass/indicator-remindor

sudo apt-get update

sudo apt-get install indicator-remindor

Install Remindor Shell

Gnome Shell extension for Gnome 3.10 & 3.12

wget https://github.com/bhdouglass/remindor-shell/archive/master.zip

unzip master.zip

mkdir -p ~/.local/share/gnome-shell/extensions/

mv ~/.local/share/gnome-shell/extensions/remindor-shell@bhdouglass.com/
