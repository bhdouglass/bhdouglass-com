Weather at a glance on your wrist! Simply Light is a highly customizable watchface for both Pebble and Pebble Time smartwatches. Development of Simply Light takes place in C for the watchface and JavaScript for the application logic with a configuration page built on AngularJS.

Clickable

Clickable is a replacement for the bulky Ubuntu SDK and provides an easy cli interface for building and deploying Ubuntu Touch apps. Clickable can take many different types of apps and compiles and packages them with a few keystrokes. It has greatly helped me reduce my development time on new Ubuntu Touch apps.

Source

uApp Explorer CLI

uApp Explorer CLI is a command line browser for uApp Explorer. It has many of the same filtering options available in the website. uApp Explorer CLI is built in NodeJS and is packaged as a snap. As it was my first experience with packaging snaps I was prompted to write a tutorial on packaging NodeJs apps as snaps.

ownCloud Files Scope

The first Ubuntu Touch program to integrate ownCloud access into the mobile operating system. The ownCloud Files Scope supports browsing files and previewing many file types. This scope is written with the NodeJS scopes library.

Rockwork

Rockwork is a collaborative effort to support Pebble smartwatches on the Ubuntu Touch operating system. It already supports most of the features of the official Android and iOS apps plus some extras! Rockwork is a daemon written in C++/Qt with a QML client app.

Falcon

Falcon is the first alternative app launcher for the Ubuntu Touch OS. It current supports more customization than the built-in launch provides. It was developed in order to give users more options for their phones. This scope uses the Go Unity Scopes bindings.
